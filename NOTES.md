### Spring Cloud Stream, c'est quoi ?

Spring Cloud Stream est un framework permettant de créer des applications de microservices pilotées 
par messages. Basé sur Spring Boot, il permet de développer des applications Spring autonomes et 
robustes pour la production. Il offre une configuration prédéfinie des middlewares de différents 
fournisseurs comme par example Apache Kafka.

### Apache Kafka, c'est quoi ?

Apache Kafka est une plateforme de flux d'événements distribuée et open source développée par la 
fondation Apache Software Foundation. Créée initialement par LinkedIn, elle a été rendue open source 
début 2011. Kafka est conçu pour gérer les flux de données en temps réel, permettant aux organisations 
de construire des pipelines de données robustes, évolutifs et tolérants aux pannes.
